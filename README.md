# Mimizuku

*ミミズク: 羽角(耳のような羽)を持つフクロウ。メニューバーに静かに常駐して「聞いて」いる。*

Mimizuku は、録音・オンデバイス文字起こし・ノートを統合した macOS のメニューバー常駐アプリです。マイクとシステム音声(いずれか、または両方)を録音し、Apple の Speech フレームワーク(SpeechAnalyzer / SpeechTranscriber)で完全オンデバイスのリアルタイム文字起こしを行います。データは Mac の外に出ません。

主な機能(開発中のロードマップは `docs/plan/IMPLEMENTATION_PLAN.md`):

- マイク / システム音声の録音とリアルタイム文字起こし(ライブ議事ログ)
- 3 ペインのメインウィンドウ: セッション一覧(タグ・ステータス・フィルタ)/ 波形付き再生 + 文字起こし(再生位置ハイライト・速度変更)/ 要約・TODO・メモのサブペイン
- リアルタイム翻訳(原文と訳文のセット表示)と要約・TODO 抽出(Apple のオンデバイス AI で開始)
- アプリ内辞書による認識精度の改善、録音ファイルからの再文字起こし
- 全セッション横断の検索、動画・音声ファイルのインポート

## 要件

- **macOS 26(Tahoe)以降** ―― 文字起こし API がそれ以前の OS に存在しません(docs/adr/0001)。
- **Apple Silicon** Mac。
- オンデバイス音声モデルアセット用に約 0.5〜1 GB のディスク(初回起動時に Apple のAssetInventory 経由でダウンロード)。

## 権限

| 権限 | 用途 | 要求タイミング |
|---|---|---|
| マイク | 自分側の発話を文字起こし | 初回録音時 |
| システム音声録音 | 相手側(通話・メディア)を文字起こし | 初回録音時 |

すべての処理はオンデバイスです。音声・文字起こしデータを伴うネットワーク通信は行いません。

## 話者ラベル ―― 既知の制約

話者 diarization はありません。「自分」= マイクストリーム、「相手」= システム音声ストリーム(リモート参加者は混在)です。

## 法的な注意

会話の録音前に、あなたの管轄・組織における録音同意の法令・ポリシーを遵守する責任は利用者にあります。
