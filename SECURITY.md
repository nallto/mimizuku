# セキュリティポリシー

脆弱性を発見した場合は、公開 Issue にせず、GitHub の Private Vulnerability Reporting (リポジトリの Security タブ →「Report a vulnerability」)から報告してください。

- 初回応答の目安: 5 営業日以内(ベストエフォート。個人の OSS プロジェクトです)。
- サポート対象バージョン: 最新リリースのみ。

Mimizuku は音声と文字起こしを完全にオンデバイスで処理し、それらのデータをネットワークに送信しません。したがって主要なセキュリティ面はローカルです ―― 保存セッションのファイル権限、および署名・notarize されたリリース成果物の完全性。署名証明書・プロビジョニングプロファイル・録音音声・文字起こしは決してコミットしないでください。
