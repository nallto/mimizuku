# ADR-0002: ライセンスは Apache-2.0

- ステータス: Proposed(有資格の専門家によるレビュー待ち ―― Accepted になる前にリポジトリを公開しない)
- 日付: 2026-07-10

## Context(背景)

本プロジェクトは、受託開発企業の VP でもある個人が OSS として公開する。ライセンス本文そのものより重要な、ライセンス周辺の 2 つのリスクがある。

1. **雇用・IP 帰属**: 著者の雇用契約や会社の職務発明規程が副業コードの所有権を主張しないか。ライセンス選択とは独立に、初回の公開 push 前に必ず確認する。
2. **特許エクスポージャー**: 音声捕捉 + 文字起こしは特許が活発な領域。明示的な特許許諾を持つライセンスは、企業導入者の摩擦を減らし、コントリビュータの立場を明確にする。

## Decision(決定)

Apache License 2.0。第三者帰属が生じたら `NOTICE` ファイルを追加。コントリビューションはinbound=outbound モデル(DCO サインオフ)で受け付け、当面 CLA は設けない。

## Consequences(結果)

- (+) 明示的な特許許諾と終了条項。企業ユーザーが OSS ツールを評価する際の事実上の安全牌。
- (+) 想定するほぼ全依存と両立(Apple サンプルコード派生スニペットは各ライセンスヘッダに従い帰属表示が必要)。
- (−) MIT よりやや重い(NOTICE 対応、ファイル毎ヘッダは推奨・必須ではない)。

## Alternatives considered(検討した代替案)

- **MIT**
  最もシンプルで最大限寛容だが、明示的な特許許諾がない。専門家が好むなら許容できるフォールバック。
- **GPL-3.0**
  派生の公開を保証するが、会議文字起こしツールを最も使いそうな層(企業)の採用を抑制する。却下。
